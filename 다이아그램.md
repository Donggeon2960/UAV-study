```mermaid

flowchart TD
    Start((시작)) --> Action[사용자 앱 조작/탐색]
    Action --> CheckInput{입력 감지?}
    
    CheckInput -- Yes --> Reset[타이머 초기화] --> Action
    CheckInput -- No (10초 경과) --> CalcScore[혼란 점수 계산]
    
    CalcScore --> CheckScore{임계치 초과?}
    CheckScore -- No --> Action
    CheckScore -- Yes --> IdentifyContext[현재 화면/상황 식별]
    
    IdentifyContext --> ShowPopup[맞춤형 안내 팝업 & 음성(TTS) 제공]
    
    ShowPopup --> UserResponse{사용자 반응}
    
    UserResponse -- 가이드 대로 버튼 클릭 --> ClosePopup[팝업 닫기] --> NextScreen[다음 화면 이동] --> End((종료))
    UserResponse -- 팝업 닫기 클릭 --> ClosePopupOnly[팝업 닫기 & 감지 재시작] --> Action
    
    subgraph Exception [예외: 반복 클릭]
    Action -- 동일 버튼 3회 클릭 --> MistakeGuide[클릭 실수 방지 가이드 표시]
    end
```
