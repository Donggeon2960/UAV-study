### I 게인 튜닝 가이드 (적분 제어)

```mermaid
graph LR
    %% 스타일 정의 (P 게인과 동일한 테마 유지)
    classDef root fill:#fff3e0,stroke:#ef6c00,stroke-width:3px;
    classDef good fill:#e8f5e9,stroke:#2e7d32,stroke-width:2px,align:left;
    classDef bad fill:#ffebee,stroke:#c62828,stroke-width:2px,align:left;
    classDef guide fill:#e3f2fd,stroke:#1565c0,stroke-width:2px,align:left;

    %% 중심 노드
    Main((I 게인<br/>증가)):::root

    %% 상세 노드
    Pro["<b>✅ 장점 (Pros)</b><br/>• 정상상태 오차 완전 제거 🎯<br/>• 외란(바람) 보상 능력 🌬️"]:::good
    
    Con["<b>❌ 단점 (Cons)</b><br/>• 오버슈트(넘침) 증가 📈<br/>• 진동 주기가 길어짐 〰️<br/>(반응이 둔해질 수 있음)"]:::bad
    
    Method["<b>🎯 튜닝 순서 (How-to)</b><br/>1. <b>반드시 P 튜닝 완료 후 시작</b><br/>2. I=0에서 시작 (Ki=0)<br/>3. 미세 오차가 남으면 천천히 증가<br/>4. 오버슈트가 커지면 즉시 중단 🛑<br/>--------------------------------<br/>💡 예시: Ki = 0 → <b>0.05</b>"]:::guide

    %% 연결 구조
    Main --> Pro
    Main --> Con
    Main -.->|보완책| Method
